Data Engineering project - Taiwan Stock Market Data Analysis - From Crawler, ETL to Dashboard

Posted on Sep 20, 2023

Product Architecture

Data Engineering

Under distributed architecture - Dockre swarm - Docker Portainer, using Python, Airflow, rabbitMQ, flower, and celery, to collect the daily data from Taiwan Stock Exchange, Taipei Exchange, and Taiwan Futures Exchange.

Using docker, fast API to set up RESTful API service.

Using docker swarm to set up distributed service, including web crawler, API, MySQL database, and RabbitMQ.

Using Cloud service - Linode, to manage multiple distributed machines.

Product CI/CD

Unit test, including crawler, and API.

Using CI/CD to conduct Continuous integration and deployment, and using GitHub-ci with API service.

Product Monitoring

Using Gafana & Prometheus to Maintenance and monitoring the API and crawler.

Product 1. Taiwan stock market data API

GET : API

Regardless of the program, you can download data through the API, or you can download data directly from the website. Once you have the data, you can perform statistical analysis, regression analysis, time series analysis, machine learning, and deep learning.

API Parameter

         |      Parameter      |      Description     |       Example      |
         |---------------------|----------------------|--------------------|
         |       stock_id      |   code of the stock  |        2330        |
         |      start_date     |      start time      |     2023-08-01     |
         |       end_date      |       end time       |     2023-08-31     |

Product 2: Taiwan stock market Visualize analysis Dashboard

GET : Dashboard

Using Redash to conduct SQL queries and create personalized stock market analysis dashboard.
