Data Engineering project - Taiwan Stock Market Data Analysis - From Crawler to Dashboard

Posted on Sep 20, 2023

Product Architecture

Data Engineering

Using distributed technology, rabbitMQ, flower, and celery, to collect the data from Taiwan Stock Exchange, Taipei Exchange, and Taiwan Futures Exchange.

Using docker, fast API to set up RESTful API service.

Using docker swarm to set up distributed service, including web crawler, API, MySQL database, and RabbitMQ.

Using Cloud service - Linode, to manage multiple distributed machines.

Product iteration

Unit test, including crawler, and API.

Using CI/CD to conduct Continuous integration and deployment, and using GitHub-ci with API service.

API Product

Using No-IP as domain name service provider.

Using Traefik as reverse proxy and load balancer provider.

Visulize Dashboard

Using Redash to conduct SQL queries and create personalized stock market analysis dashboard.
